Ejemplos Visuales de HTML y CSS

Este documento muestra cómo funciona cada parte del HTML y CSS junto con su visualización en la página.

1. Estructura Básica de HTML

Esta es la estructura mínima de un documento HTML. La etiqueta <!DOCTYPE html> indica que se utiliza HTML5. 

<!DOCTYPE html>
<html>
  <head>
    <title>Título de la Página</title>
  </head>
  <body>
    <h1>Encabezado Principal</h1>
    <p>Este es un párrafo de ejemplo.</p>
  </body>
</html>

2. Formularios

El siguiente ejemplo muestra un formulario simple. La etiqueta <form> contiene los elementos del formulario. La etiqueta <label> se asocia al campo de entrada (<input>) mediante el atributo for. 

<form action="/submit">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre">
  <input type="submit" value="Enviar">
</form>

3. Tablas

Las tablas se utilizan para organizar datos en filas y columnas. La etiqueta <table> define la tabla, <tr> define cada fila, <th> define los encabezados y <td> los datos. 

<table border="1">
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>23</td>
  </tr>
  <tr>
    <td>Juan</td>
    <td>30</td>
  </tr>
</table>

4. CSS Básico

CSS (Cascading Style Sheets) se utiliza para dar estilo a los elementos HTML. En este ejemplo se muestra cómo aplicar estilos de forma interna. 

<style>
  body {
    background-color: lightgray;
  }
  h1 {
    color: blue;
  }
</style>
